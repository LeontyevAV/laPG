# laPG

## Архитектура

- Главный модуль: `main.py` (меню 7 пунктов)
- Функциональные модули: `backup.py`, `restore.py`, `db_utils.py`, `settings.py`
- `backup.py` и `restore.py` можно запускать отдельно (с CLI-флагами)
- Взаимодействие с PostgreSQL через CLI-утилиты (`pg_dump`, `pg_restore`, `psql`, `createdb`, `dropdb`) — `subprocess`
- `asyncpg` используется только для получения списка БД с сервера

## Команды

```bash
# Главное меню
python main.py

# Бекап (интерактивный выбор БД)
python backup.py
# Бекап всех БД параллельно
python backup.py --all

# Восстановление
python restore.py
```

## Конфигурация

**`.env`** — только секреты:
```
DB_PASSWORD=your_password
```

**`settings.yaml`** — все остальные настройки:
```yaml
connection:
  host: localhost
  user: postgres

backup:
  default_compress: 0
  default_keep: 0
  databases:
    labots:
      compress: 9
      keep: 20

restore:
  backup_dirs:
    - backup
    - restory

scheduler:
  enabled: false
  cron: "0 2 * * *"
```

`.env` имеет приоритет над `settings.yaml` (DB_HOST/USER из `.env` переопределяют `connection`).

## Структура

- `main.py` — главное меню (8 пунктов)
- `backup.py` — создание дампов (интерактивный и `--all` режимы)
- `restore.py` — восстановление из дампа
- `cron.py` — управление cron (чтение/запись crontab, перезапуск)
- `db_utils.py` — общий модуль: получение списка БД, ротация бекапов
- `settings.py` — загрузка и модель конфигурации (pydantic + YAML)
- `settings.yaml` — конфигурация (пути, ID, флаги, списки)
- `.env` — секреты (пароль)
- `backup/`, `restore/` — директории с `.dump`-файлами

## Соглашения

- Дампы в формате custom (`-F c`) с именем `{db}_backup_{timestamp}.dump`
- Список БД получаем через `asyncpg` с fallback `postgres` → `template1`
- `--keep N` — удаляет старые дампы, оставляя N последних (по дате в имени файла)
- Восстановление: drop + create + pg_restore
- `backup.py`/`restore.py` импортируются из `main.py` без исполнения (guarded by `if __name__`)
- Настройки: `.env` (secrets) + `settings.yaml` (всё остальное), мерж через `settings.py`
